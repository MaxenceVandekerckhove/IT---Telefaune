Afin de lancer le projet sur votre propre navigateur en local, 

Vous trouverez dans le dossier SQL Export, un export de commandes SQL permettant la création de la base de donnée ainsi que des données initialement présentes.

Une fois ceci fait, vous devez modifier la chaine "ConnectionStrings" dans appsetting.json afin d'y integrer votre propre chaine de connexion.
